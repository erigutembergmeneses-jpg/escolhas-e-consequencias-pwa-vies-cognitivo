# PWA Viés Cognitivo

Uma Progressive Web App leve, otimizada para mobile, sem login e sem carregamento pesado.  
O objetivo é simular cenários rápidos e revelar o viés cognitivo dominante do usuário.

## 🚀 Como usar
1. Abra `index.html` em um navegador moderno.
2. A tela inicial aparece por 2 segundos.
3. Em seguida, o usuário escolhe entre 3 cenários.
4. O resultado mostra o viés dominante e sugere um capítulo.
5. Botões permitem:
   - Ler a introdução grátis.
   - Compartilhar o perfil neural (imagem minimalista via Web Share API).

## 🛠️ Tecnologias
- HTML, CSS, JavaScript puro.
- Canvas API para gerar imagem.
- Web Share API para compartilhar.
- Manifest + Service Worker podem ser adicionados para tornar a PWA instalável.
